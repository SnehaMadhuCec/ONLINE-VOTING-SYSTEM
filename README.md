# Online Voting System

A lightweight Online Voting System developed using **Java**, **JSP**, **Servlets**, **MySQL**, **HTML/CSS**, and **Apache Tomcat**. The project demonstrates the core functionalities of an electronic voting platform, including voter authentication, candidate management, vote casting, and result generation.

## Features

* User registration and login
* Candidate management
* One-time vote casting
* Vote counting and result display
* Admin dashboard

## Tech Stack

* Java
* JSP
* Servlets
* HTML/CSS
* MySQL
* Apache Tomcat
